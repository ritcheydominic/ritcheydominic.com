In May of 2022, the RIPE NCC assigned me AS208078 along with 2 IPv6 ranges (2a05:b0c6:5500::/40 and 2a06:1287:3800::/40). I began advertising this IP space using BIRD on Vultr cloud compute instances I already had in use for my family's mail server and other projects. Portions of these IPv6 ranges are advertised as an anycast network while others are separate unicast networks for each instance. I'm currently using this IP space to create an anycast WireGuard VPN where users are connected to and assigned IP addresses in the node closest to them. This utilizes Guardpost to manage each node's WireGuard interfaces.

My future plans for this space include setting up a NAT64/464XLAT translator network using this (and IPv4) space to research how this transitional solution will fare as IPv6 utilization rises and IPv4 space becomes scarcer. I will also try and test newer solutions as they are published.
